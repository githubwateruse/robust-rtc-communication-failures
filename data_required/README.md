# Generated and user-prepared data

- `failure_patterns/`: generated by `generate_failure_patterns.py`.
- `training_runs/`: generation workbooks produced by policy training.
- `simulation_outputs/`: output workbooks produced by testing scripts.

The large generated files in these directories are not part of the clean source
materials. Failure patterns can be regenerated with probabilities `0.001` and
`0.01`; all testing scripts use pattern IDs `0`-`9`.
